# fuzzy_doc_ranking

Ogawa, Yasushi, Tetsuya Morita, and Kiyohiko Kobayashi. "A fuzzy document retrieval system using the keyword connection matrix and a learning method." Fuzzy sets and systems 39.2 (1991): 163-179.
